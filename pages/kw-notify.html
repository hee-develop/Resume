Kwangwoon Notify

광운알리미

Description

Kwangwoon Notify is an unofficial application that designed to improve accessibility to various services in the university.
This provides to notify new notification via the push server.
And this includes QR code identification for library, university's schedule, telephone numbers in each department, and U-CAMPUS for check new assignment.

Used

ClientAndroid Server(parsing)Node.js on AWS Server(push)Google Firebase Cloud Messaging DBMySQL

Open Source

uCropfor crop profile image Retrofitfor using login to U-CAMPUS Jsoupfor parse XML and HTML Firebase messagingfor receive push message CircleImageViewfor display a circle image(for profile)
